Параметры администратора

Для добавления нового администратора кластера следует выбрать в дереве центральных серверов требуемый сервер, выбрать требуемый кластер, зарегистрированный на этом сервере, выбрать ветку Администраторы и выполнить команду меню или контекстного меню Добавить.

Администратор кластера имеет следующие свойства:

  • Имя - Имя администратора кластера. Уникально в пределах каждого кластера.
    Важно! Создание администратора с пустым именем запрещено.
  • Описание - Произвольное описание администратора кластера.
  • Аутентификация паролем - Признак аутентификации паролем. По умолчанию установлен.
  • Пароль - Пароль администратора кластера.
  • Подтверждение пароля - Подтверждение пароля.
