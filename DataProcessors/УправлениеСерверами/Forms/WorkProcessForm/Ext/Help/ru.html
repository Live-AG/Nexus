Параметры рабочего процесса

Чтобы просмотреть и настроить свойства рабочего процесса, в контекстном меню процесса следует выбрать команду Изменить.

Следущие свойства недоступны для редактирования:

  • Компьютер - Рабочий сервер, на котором работает процесс.
  • Лицензия - Информация о том, какая серверная лицензия используется данным рабочим процессом и характеристика этой лицензии.
  • Активен - Если флажок установлен, рабочий процесс используется и активен.
  • Резервный - Если флажок установлен, рабочий процесс является резервным.
  • Включен - Если флажок установлен, рабочий процесс включен и может использоваться системой.
  • Производительность - Относительное значение производительности.
  • Использование - Признак использования рабочего процесса. Возможные значения: Используется, Не используется, Используется как резервный.

Статистика

  • Время запуска - Время последнего запуска рабочего процесса.
  • PID процесса ОС - Номер процесса (в терминах ОС, под управлением которой выполняется рабочий процесс).
  • IP порт - Сетевой порт рабочего процесса, который выделяется системой динамически при запуске рабочего процесса из диапазона сетевых портов, указанного для данного сервера.
  • Занято памяти (КБайт) - Объем памяти, занятый рабочим процессом.
  • Превышение над критическим значением (сек) - Время (в секундах) превышения параметра Занято памяти над параметром Допустимый объем памяти кластера серверов.
  • Доступная производительность - Текущая доступная производительность.
  • Затрачено СУБД - Среднее время, затраченное СУБД на обслуживание одного соединения.
  • Реакция сервера - Среднее время, затраченное на обслуживание одного соединения. Равно сумме значений следующих четырех полей: Затрачено сервером, Затрачено СУБД, Затрачено менеджером блокировок.
  • Затрачено сервером - Среднее время, затраченное рабочим процессом на обслуживание одного соединения.
  • Затрачено менеджером блокировок - Среднее время, затраченное менеджером блокировок на обслуживание одного соединения.
  • Клиентских потоков - Среднее количество клиентских потоков, обработанное рабочим процессом. Используется системой для вычисления производительности рабочего процесса.
