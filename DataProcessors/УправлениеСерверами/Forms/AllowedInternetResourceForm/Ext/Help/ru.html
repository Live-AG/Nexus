Параметры разрешенного интернет-ресурса

Чтобы создать новый разрешенный интернет-ресурс, необходимо в обработке открыть контекстное меню ветки Профили безопасности - Ресурсы интернет и выбрать пункт Добавить. Для того чтобы изменить параметры интернет-ресурса, следует в контекстном меню ресурса Интернет выбрать команду Изменить.

При редактировании существующего интернет-ресурса свойство Имя недоступно для изменения.

Разрешенный ресурс сети Интернет описывается следующими параметрами:

  • Имя ‑ имя ресурса для его идентификации. Уникально в пределах профиля безопасности.
  • Адрес ‑ адрес ресурса без указания протокола.
  • Порт ‑ номер порта, через который выполняется взаимодействие с указанным ресурсом.
  • Протокол ‑ протокол, по которому выполняется взаимодействие с ресурсом. Могут быть указаны следующие протоколы (регистр символов не имеет значения):
    • imap ‑ сервер электронной почты, работающий по протоколу IMAP.
    • pop3 ‑ сервер электронной почты, работающий по протоколу POP3.
    • smtp ‑ сервер электронной почты, работающий по протоколу SMTP.
    • http ‑ веб-сервер.
    • https ‑ защищенное соединение с веб-сервером.
    • ftp ‑ ftp-сервер.
    • ftps ‑ защищенное соединение с ftp-сервером.
